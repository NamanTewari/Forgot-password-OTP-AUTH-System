# Forgot Password OTP Auth System

## Features
- User Registration
- User Login
- JWT Authentication
- Cookie Based Authentication
- Forgot Password
- OTP Verification
- Password Reset
- Nodemailer Email Integration
- MongoDB Database
- MVC Architecture

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Nodemailer
- EJS
- bcrypt

## Installation
npm install
npm start
